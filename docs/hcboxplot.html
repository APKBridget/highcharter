Function to make a boxplot

Usage

hcboxplot(x, by = NULL, outliers = TRUE, horizontal = TRUE, ...)

Arguments

x
A numeric vector.
by
A string vector same length of x.
outliers
A boolean value to show or not the outliers.
horizontal
A boolean value indicating if the boxplots should be horizontal.
...
Aditional arguments for the data series (http://api.highcharts.com/highcharts#series).

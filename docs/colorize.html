Create vector of color from vector

Usage

colorize(x, colors = c("#440154", "#21908C", "#FDE725"))

Arguments

x
A numeric, character or factor object.
colors
A character string of colors (ordered) to colorize x

Description

Create vector of color from vector

Examples


colorize(runif(10))


 [1] "#CCD33B" "#399980" "#3C2060" "#440154" "#2C6079" "#248085" "#34406C" "#6AAD69" "#9BC052"
[10] "#FDE725"




colorize(LETTERS[rbinom(20, 5, 0.5)], c("#FF0000", "#00FFFF"))


 [1] "#FF0000" "#FF0000" "#AA5555" "#AA5555" "#AA5555" "#AA5555" "#FF0000" "#AA5555" "#00FFFF"
[10] "#AA5555" "#55AAAA" "#55AAAA" "#AA5555" "#00FFFF" "#55AAAA" "#FF0000" "#55AAAA" "#55AAAA"
[19] "#55AAAA"

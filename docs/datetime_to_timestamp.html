Date to Timesstamps

Usage

datetime_to_timestamp(dt)

Arguments

dt
Date or datetime vector

Description

Turn a date time vector to timestamp format

Examples


datetime_to_timestamp(
  as.Date(c("2015-05-08", "2015-09-12"),
  format = "%Y-%m-%d"))
